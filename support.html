Troubleshooting And Launch Help

Common Customer Questions

  • If a video has no captions, transcript loading and transcript export are unavailable.
  • Markdown export is part of the free plan. CSV, JSON, and transcript export require Pro.
  • If a Pro user changes devices, they should deactivate the old device first or use the Lemon Squeezy customer portal if it is enabled.

Before Store Submission

  • Keep the GitHub Pages support URL in config.js and only replace the billing values that still depend on Lemon Squeezy.
  • Build and publish the public docs site from site-src/.
  • Use the GitHub support issue form as the temporary intake path until Help Scout is live.
  • Publish the privacy policy from store/PRIVACY_POLICY.md to the Pages URL in config.js.
  • Follow LAUNCH_STACK_SETUP.md and store/RELEASE_CHECKLIST.md.

This local support page is a safe fallback, but the release listing should point to the public GitHub Pages support URL.
