Privacy

Data Use Summary

This extension is designed to work locally first. It accesses only the information needed to turn a captioned YouTube video into study materials.

What The Extension Reads

  • The current YouTube watch page URL, title, author, description, and available caption tracks.
  • Caption text for the video only after the user clicks Load Transcript.
  • Saved study packs stored in Chrome on the local device.

What The Extension Stores

  • Generated summaries, flashcards, quizzes, and short transcript excerpts in chrome.storage.local.
  • Onboarding state and local Pro-license state for the current browser profile.

When Data Leaves The Extension

  • If Chrome's built-in Prompt API is available, study-pack generation can use that on-device capability.
  • If a user activates, refreshes, or deactivates a Pro license, the extension contacts Polar's public license API with the provided email and license key.
  • This launch build does not send analytics events or error reports to third-party telemetry services.

What It Does Not Do

  • No ad tracking or browsing-history collection is included.
  • No microphone, webcam, or non-YouTube page access is requested.

Before publishing to the Chrome Web Store, publish the final policy from store/PRIVACY_POLICY.md through GitHub Pages and place that URL in config.js.
