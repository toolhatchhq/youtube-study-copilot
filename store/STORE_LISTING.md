# Chrome Web Store Listing Draft

## Name

YouTube Study Copilot

## Short Description

Turn YouTube lessons into notes, flashcards, quizzes, and exportable study packs.

## Detailed Description

YouTube Study Copilot helps students, language learners, and self-learners turn captioned YouTube videos into practical study materials inside Chrome.

Use it to:

- load captions from the current YouTube watch page
- generate concise notes and key takeaways
- turn important concepts into flashcards
- create quiz prompts for review sessions
- save your best study packs locally
- export study packs as Markdown, CSV, JSON, or transcript text

Free plan:

- generate study packs
- save the latest 5 packs locally
- export Markdown

Pro plan:

- export flashcards as CSV
- export complete packs as JSON
- export transcripts as TXT
- keep a deeper local archive
